Pushing Your Blog to Github And Viewing as Github pages

Fri, Jan 8, 2016

IMP :- This post is based on the windows version of hugo.

You may visit Link For the repository

In Our last post, we had done making the blog. Now we will be pushing it to Github. To do this you must have a little knowledge about using Git and Github.

First Create a new repository on your account.Once done open your cmd and set your location to a folder named public where your blog was created . Now type the following commands.

                                            git init

                                            git checkout --orphan gh-pages                                              

                                            git add .

                                            git commit -m "Initial Commit"

                                            git remote add origin [YOUR_REPOSITORY_LINK]     { For Example :- git remote add origin https://github.com/codein-dwij/blog_hugo.git }

                                            git push origin gh-pages

Now enter your account details and press enter. Your blog is successfully pushed to Github. To View Your blog , type on the address bar

                                            http://YOUR_USERNAME.github.io/REPOSITORY_NAME   { For Example :- http://codein-dwij.github.io/blog_hugo}

You may see your blog generated there. You can add more posts to your blog , generate it using “hugo –” command and push it to github!
